# CT-EUS Vessel Pipeline v0.1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build an isolated, testable baseline project for CT-EUS vessel segmentation and reconstruction without touching the existing `location2EUS` or old vessel reconstruction projects.

**Architecture:** The baseline separates configuration, DICOM indexing, phase classification, HU candidate thresholding, geometry post-processing, iteration documentation, and CLI orchestration. Ground-truth labels are represented only as an evaluation path in configuration and are guarded from work/output paths.

**Tech Stack:** Python 3.12, mamba/conda environment, pydicom, PyYAML, NumPy, SciPy, scikit-image, SimpleITK, pytest, TotalSegmentator as planned external preprocessing.

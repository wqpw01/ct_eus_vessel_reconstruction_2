# CT-0021 v0.2-phase-inference 迭代记录

生成时间: 2026-06-24

## 数据使用红线

Ground Truth 仅用于最终评价，不参与分割、阈值估计、候选生成、训练、后处理或参数搜索。本轮只读取原始 DICOM header 与既有输出索引，不读取人工标签。

## Pipeline 摘要

- 修正 CT-0021 多期增强 CT 的期相逻辑。
- 对连续 `1.0 x 1.0_A` 原始增强薄层序列按 `SeriesNumber` 做时序推断。
- 保留 `metadata_phase` 字段记录 DICOM 文字推断结果。
- 新增 `dynamic_phase` 字段记录时间动态推断结果。
- 最终 `phase` 优先采用 `dynamic_phase`，因此最后一个连续增强序列可被识别为 venous。

## 核心参数

```yaml
dynamic_series_description_rule: "contains 1.0 and _A"
excluded_dynamic_keywords:
  - mip
  - scout
  - dose
  - lung
  - med
  - report
min_dynamic_instances: 100
ordering_key:
  - series_number
  - series_uid
phase_assignment:
  first_dynamic_series: arterial
  middle_dynamic_series: portal
  last_dynamic_series: venous
ground_truth_usage: evaluation_only
```

## 已知局限

- 本轮仍未执行完整血管分割、TotalSegmentator 剔除、配准、多期融合或三维 mesh 重建。
- 门静脉最佳序列仍需下一轮结合图像 HU 增强强度、Frangi/Hessian 管状响应与连通性评分自动确认。
- 如果未来病例存在不同命名格式，动态序列识别规则需要扩展为基于 DICOM acquisition time、series time 和增强序列空间覆盖的一般化策略。
